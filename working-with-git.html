Working with git

1. add files to your project

this can be done via the touch command or through the interface as usual

Example: $ touch index.html && touch main.css

2. Add new or modified files to staging

Once we add new files or modify existing files we can tell git to track them by adding them to staging for next commit (think of commits as checkpoints). This will let git know what files the next checkpoint will consist of.

$ git add < filename >

or

$ git add .
